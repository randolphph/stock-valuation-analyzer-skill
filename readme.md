# Stock Valuation Analyzer

一个用于计算**净现金**和**扣除净现金后真实 PE**的 Claude Skill，帮你看清一家公司经营业务的真实估值。

> 报表 PE 告诉你市场给整个公司标了什么价。真实 PE 告诉你，你为经营业务本身付了多少钱。

## 为什么需要这个

A 股和港股有大量账上趴着巨额现金的公司——白酒、消费品、软件平台。用报表 PE 直接看估值，会把现金和经营业务混在一起。扣除净现金后的真实 PE 才能反映你为经营业务实际支付的价格。

## 核心功能

- **净现金计算**：从资产负债表自动提取关键科目，计算净现金头寸
- **真实 PE**：扣除净现金后的 ex-cash PE，与报表 PE 对比
- **辅助指标**：现金占市值比、每股净现金、现金收益比（经营现金流/净利润）
- **可视化输出**：Summary 看板 + 瀑布图，一眼看清净现金构成
- **支持上传财报**：PDF 或 Excel 格式的年报/季报，自动提取数据

## 净现金口径

采用「负债端保守、资产端审慎」的方法论：

```
净现金 = (货币资金 + 交易性金融资产)
       + min(应收账款+应收票据, 应付账款+应付票据)
       + 合同负债
       - 总负债
```

几个关键设计选择：

- **用总负债而非有息负债**：更保守，假设所有负债都是真实义务，再选择性加回
- **min(应收, 应付)**：取两者较小值作为经营性对冲，保守计量
- **加回合同负债**：仅适用于对下游有强定价权的公司（茅台、可口可乐型护城河），合同负债本质上是无息融资，是实力的体现而非真实债务

## 使用方式

### 在 Claude.ai 中安装

下载 `stock-valuation-analyzer.skill` 文件，在 Claude.ai 设置中导入即可。

### 在 Claude Code 中安装

```bash
# 方式一：克隆到个人 skill 目录
git clone https://github.com/YOUR_USERNAME/stock-valuation-analyzer.git ~/.claude/skills/stock-valuation-analyzer

# 方式二：克隆到项目 skill 目录
git clone https://github.com/YOUR_USERNAME/stock-valuation-analyzer.git .claude/skills/stock-valuation-analyzer
```

### 触发方式

安装后，以下说法都会自动触发这个 skill：

- "帮我算一下茅台的真实 PE"
- "看看这个公司账上有多少现金"
- "分析一下这份年报的净现金"
- 直接上传一份财报 PDF 并提到"估值"或"净现金"

### 输入数据

**上传财报**（推荐）：上传 PDF 或 Excel 格式的年报/季报，skill 会自动从资产负债表、利润表、现金流量表中提取所需数据。

**手动提供**：在对话中直接给出关键数据，至少需要：

| 科目 | 说明 |
|------|------|
| 货币资金 | 必须 |
| 交易性金融资产 | 必须（无则填 0） |
| 应收票据 + 应收账款 | 必须 |
| 应付票据 + 应付账款 | 必须 |
| 合同负债 | 必须 |
| 负债合计 | 必须 |
| 归母净利润 | 必须 |
| 总股本 | 必须 |
| 股价或总市值 | 必须 |
| 经营活动现金流净额 | 可选（用于计算现金收益比） |

## 适用范围与限制

**适用**：A 股和港股上市的非金融类企业，尤其是账上现金充裕的消费品、科技平台类公司。

**不适用**：
- 银行、保险、券商等金融企业（资产负债表结构完全不同）
- 亏损企业（PE 无意义，仅输出净现金分析）
- 合同负债代表真实交付义务的企业（如软件、工程施工），此时不应加回合同负债
